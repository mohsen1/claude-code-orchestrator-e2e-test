USAGE INSTRUCTIONS
================================================================================

1. Copy environment variables:
   cp .env.example .env.local

2. Generate secrets:
   openssl rand -base64 32  # for NEXTAUTH_SECRET

3. Install dependencies:
   npm install

4. Run development server:
   npm run dev

5. Build for production:
   npm run build

6. Docker deployment:
   docker-compose up --build

7. Run tests:
   npm run test

8. Type checking:
   npm run typecheck

9. Linting:
   npm run lint
